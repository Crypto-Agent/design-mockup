Crypto-Agent
채팅으로 끝내는 학회 AI Agent

흩어진 온·오프체인 데이터를 한국어 자연어로 통합 조회·분석·실행하는 LLM 기반 멀티 에이전트. 메인 오케스트레이터가 의도를 라우팅해 Sub-Agent를 호출하고, 채팅 + 이해 캔버스로 출처 기반 결과를 점진 전달합니다.

03

C안 디자인 시안 — 트랜잭션 한국어 설명

Phase 3 / KBW 2026.09 데모 기능. Sub-Agent와 대화하며 이해 캔버스(요약·사실 → 자금흐름 → 개념 → 출처·안전)가 누적되는 흐름을 구현한 인터랙티브 시안.

🗣️ 채팅 중심

독립 페이지가 아니라 메인 에이전트가 호출하는 Sub-Agent. 호출 시각화 + 대화로 진행.

🧩 이해 캔버스

대화 턴마다 화면이 누적 — 요약·사실 / 자금흐름 / 개념(소크라테스) / 출처·안전.

🛡️ 신뢰 · 한국어

출처 4건 인용, 스캠 대조, 가상자산법 고지. 단정 대신 스스로 점검 질문.
